Smartwatch API docs
  • A BEARER AUTH TOKEN IS REQUIRED TO PERFORM THE FOLLOWINGS ACTIONS !
  • http://localhost:8080/smartwatch/api/auth
GET
/datas

Get all the Smartwatch statement

  • All codes

  • Code 200 : Success, return an application/json
  • Code 500 : Internal Server Error
  • A BEARER AUTH TOKEN IS REQUIRED !
GET
/datas/{id}

Get a Smartwatch statement, using its id

  • All codes

  • Code 200 : Success, return an application/json
  • Code 404 : SmartwatchData statement not found
  • Code 500 : Internal Server Error
  • A BEARER AUTH TOKEN IS REQUIRED !
POST
/datas

Add a SmartwatchData statement

  • All codes

  • Code 200 : Success, return an application/json
  • Code 400 : Bad Request
  • Code 500 : Internal Server Error
  • A BEARER AUTH TOKEN IS REQUIRED !
PUT
/datas/{id}

Update an SmartwatchData statement

  • All codes

  • Code 200 : Success, return an application/json
  • Code 400 : Bad Request
  • Code 500 : Internal Server Error
  • A BEARER AUTH TOKEN IS REQUIRED !
DELETE
/datas/{id}

Delete a SmartwatchData statement by its id

  • All codes

  • Code 200 : Success, return an application/json
  • Code 500 : Internal Server Error
  • A BEARER AUTH TOKEN IS REQUIRED !
